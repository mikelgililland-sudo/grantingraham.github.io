# Site Governance (grantingraham.me archive)

- One H1 per page/post.
- Use btn-primary / btn-secondary only for CTA links.
- Keep WordPress export XMLs in /wordpress-export.
- Posts archive path: content/posts/YYYY/MM/SLUG/
- Pages archive path: content/pages/SLUG/
- Each post/page folder contains:
  - index.html (canonical content)
  - notes.md (editor notes)
  - sources.md (references)
